# Phidgets_Thumbstick_CMU
Demonstration of using Desktop CMU graphics and Phidgets 

# Instructions
To test out this game:
1. Download the script - Thonny is recommended IDE
2. Install CMU Graphics library - cmu-graphics
3. Install the Phidgets library - Phidgets22Native
4. Obtain a getting started kit from Phidgets [https://www.phidgets.com/education/learn/getting-started-kit-tutorial/]
5. Obatin a Thumbstick Phidget [https://www.phidgets.com/?prodid=962]
6. Have fun!!


Learn more about using Phidgets with CMU [https://www.phidgets.com/education/learn/projects/cmu]
